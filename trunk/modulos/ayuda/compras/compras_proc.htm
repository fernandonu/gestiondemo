A - Ordenes Pendientes
Haciendo click sobre una de las ordenes se obtienen los datos de esa orden; se pueden modificar los datos del cliente, la forma de pago, añadir o eliminar productos. Luego de modificar los datos presionar el botón Guardar.
También puede anular la orden (presionando el botón Anular) o presionando el botón Para autorizar enviar la orden para que esta sea autorizada.

B - Ordenes Para Autorizar
Haciendo click sobre una de las ordenes se obtienen los datos de esa orden; con esto se permite autorizar la orden (presionando el botón Autorizar), también puede anularse o rechazarse la orden (con lo cual la orden vuelve al estado de "Pendiente")
Hay una opción de Resumen con la cual se envia un mail con el resumen de los pagos.

C - Ordenes Autorizadas
Haciendo click en el box de la orden seleccionada se habilita la opción Enviar mail.
Haciendo click sobre una de las ordenes se obtienen los datos de esa orden; presionando el botón Forma de Pago se permite cambiar la forma de pago. También se puede pagar la orden, terminarla (cuando se recibieron los productos o cuando se pagan los productos) o anularla.

D - Ordenes Enviadas
Haciendo click en el box de la orden, en el campo Montos Ordenes se muestra la suma de las ordenes seleccionadas..
Haciendo click sobre una de las ordenes se obtienen los datos de esa orden.

- Si la orden esta parcialmente pagada: se puede pagar o terminar la orden. También presionando el botón Resumen de pagos se muestra el resumen de los pagos de las ordenes.
- Si la orden esta enviada: se puede anular, pagar, terminar o rechazar la orden. Con la opción Resumen de pagos se obtiene un resumen de los pagos de las ordenes. Presionando el botón Pagar se muestra una pantalla donde se pueden pagar las ordenes con la forma de pago definida. Presionando el botón Pago Múltiple se pueden agregar mas ordenes a esa forma de pago.
Con la opción Deshabilitar Pago especial se muestra un cartel "Se deshabilitó la forma de pago especial para la orden...".Se vuelve a selecionar la orden; esta opción no permite cambiar la forma de pago.
Con la opción Habilitar Pago especial se muestra un cartel "Se habilitó la forma de pago especial para la orden...".Se vuelve a selecionar la orden; esta opción permite cambiar la forma de pago.
- Si la orden esta marcada con pago múltiple: se puede anular, pagar, terminar o rechazar la orden. Con la opción Resumen de pagos se obtiene un resumen de los pagos de las ordenes. Presionando el botón Pagar se muestra una pantalla donde se pueden pagar las ordenes con la forma de pago definida. Presionando el botón Pago Múltiple se pueden agregar mas ordenes a esa forma de pago.
Con la opción Desabilitar Pago especial se muestra un cartel "Se deshabilitó la forma de pago especial para la orden...".Se vuelve a selecionar la orden; esta opción no permite cambiar la forma de pago.
Con la opción Habilitar Pago especial se muestra un cartel "Se habilitó la forma de pago especial para la orden...".Se vuelve a selecionar la orden; esta opción permite cambiar la forma de pago.

E - Ordenes Pagadas
Haciendo click en el box de la orden, en el campo Montos Ordenes se muestra la suma de las ordenes seleccionadas y en el campo Montos Pagados se muestra el importe que ha sido pagado del total.
Haciendo click sobre una de las ordenes se obtienen los datos de esa orden.
- Si la orden esta parcialmente pagada: se puede pagar o terminar la orden. También presionando el botón Resumen de pagos se muestra el resumen de los pagos de las ordenes. Presionando el botón Forma de pago se puede cambiar la forma de pago.
- Si la orden esta totalmente pagada : se puede terminar la orden. Con la opción Resumen de pagos se obtiene un resumen de los pagos de las ordenes.
- Si la orden esta marcada con pago múltiple: se puede terminar la orden. Con la opción Resumen de pagos se obtiene un resumen de los pagos de las ordenes.
