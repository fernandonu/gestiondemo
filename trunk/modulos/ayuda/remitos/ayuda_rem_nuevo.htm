Esta página permite llenar los datos del remito y añadir los productos correspondientes al remito.

Si el remito pertenece a una factura ingresar el número de factura y presionar el botón Buscar. Esto carga los datos del cliente de la factura en el remito y los productos que se encuentran en la factura. Por defecto se asocia la factura al remito. Si no se debe asociar la factura, dejar sin seleccionar el botón de chequeo Asociar.
El botón Ver factura muestra el detalle de la factura.

Si no existe la factura para el remito
Completar los datos que se solicitan en la página:

1) Presione el botón Elegir cliente. Esto mostrará una ventana con la lista de clientes existentes y un formulario para completar los datos del cliente.
 2) En el caso que no se encuentre en la lista el cliente puede agregar un nuevo cliente.
   2.1) Presione el botón Nuevo.
  2.2) Complete los datos y presione el botón Guardar.
 3) .Seleccione el nombre del cliente . Presione el botón Elegir Cliente para cargar el cliente elegido.
 4) Presione el botón Cerrar.

Completar los datos del remito.
Tildar el checkbox con titulo “Imprimir con precios” para que el archivo pdf que muestra el remito se imprima con los precios para cada producto. Si el botón queda sin chequear el archivo se muestra sin precios.

Presionar el botón Buscar. Esto muestra las licitaciones existentes. Seleccionar una licitación. Esto asocia la licitación al remito. Presionar el botón Ir para ver los datalles de la licitación.Si no se debe asociar la licitación, dejar sin seleccionar el botón de chequeo.

Para añadir productos al remito
1) Presione el botón Agregar.
 2) Seleccione el tipo de producto. Esto mostrará los productos cargados para el tipo de producto seleccionado.
3) Seleccione un producto
4) Presione el botón Cargar.
5) Repitir desde el paso 2 hasta el paso 4, mientras se deba cargar productos.
6) Presionar el botón Salir.

Si presiona el botón Guardar se guardarán los datos y productos que se hayan ingesado hasta el monento y luego al recuperar el remito desde Inicio/Aministracion /Remitos/ Listar remitos se podrán añadir más productos al remito. Esto cambiará el estado del remito a Pendiente.

Si presiona el botón Terminar se termina el remito. Esto cambiará el estado del remito a "terminado" y no se permiten ingresar nuevos productos al remito. Además se generará un archivo en formato pdf que se puede ver desde Inicio/Administracion/ Remitos/Listar Remitos.

En caso de que el remito se crea sin una factura se manda un mail avisando que se ha creado un remito sin factura. .
