Para modificar el renglón que esta seleccionado
1) Seleccionar un competidor para el renglón, si el renglón fue adjudicado al competidor seleccionado, chequear el botón ( ) con título Adjudicada.
2) Seleccionar el tipo de moneda del renglón.
3) Ingresar el monto unitario y total del renglón.
4) Presionar el botón Aceptar.

Para modificar varios renglones
1) Presionar el botón Buscar renglones.
2) Marcar el renglón que se va a modificar.
3) Seleccionar un competidor para el renglón, si el renglón fue adjudicado al competidor seleccionado, chequear el botón ( ) con título Adjudicada.
4) Seleccionar el tipo de moneda del renglón.
5) Ingresar el monto unitario y total del renglón.
6) Presionar el botón Aceptar.

Agregar un renglón
Presionar el botón Agregar; este botón esta habilitado si la licitación tiene renglones. Ingresar información del renglón (código, título, cantidad) y presionar Aceptar.
