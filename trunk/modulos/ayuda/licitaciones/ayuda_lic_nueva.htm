En esta página se permite ingresar los datos para una nueva licitación.

Completar los datos que se requieren en la página y presionar el Botón Aceptar

Si entre los datos que se muestran en los selectores correspondientes para Distritos, Monedas, Entidades, Estado, Tipos de entidades no está disponible el que usted debe ingresar puede añadirlo desde Inicio/Administracion/ Licitaciones/ Configuración.
En la página de configuración se muestran los datos que ya están cargados y se permite ingresar un nuevo dato.
