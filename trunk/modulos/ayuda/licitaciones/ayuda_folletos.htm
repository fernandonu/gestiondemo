Esta página permite cargar folletos para un producto seleccionado.
1) Seleccione tipo de producto y presione el botón Agregar. Esto mostrará todos los productos y proveedores que se hayan cargado para el tipo de producto seleccionado.
2) Para agregar el producto:
  2.1) Seleccionar un producto (click en el botón de chequeo o en la fila que muestra el producto) Presionar Agregar.Esto mostrará el producto seleccionado en la página Cargar Folleto.
3) Ingresar el nombre del archivo que contiene el folleto para el producto seleccionado (o seleccionar el archivo desde el boton Examinar).
4) Presionar Guardar folleto.
