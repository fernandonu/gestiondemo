Esta página permite realizar ofertas para cada renglón de una licitación seleccionada.
1) Se muestra Número de licitación, y la entidad que ofrece la licitación.
2) Se debe ingresar el valor del dolar y Presionar Aceptar.
3) Se muestran los renglones de la licitación, con el título, ganancia, precio unitario, precio total.

Se permite Agregar un nuevo renglón, Modificar, eliminar, Ver descripciones de los productos de cada renglón y terminar la licitación.

Agregar un nuevo renglón

1) Presionar el botón Agregar Renglón. Esto muestra un formulario que permite cargar los datos del renglón.
2) Llene los datos para la información del renglón, Descripción del renglón, Adicionales para el Renglón, y puede agregar Productos Adicionales.
Para Ingresar Productos adicionales: En la tabla con títulos Productos Adicionales presionar el botón agregar, esto muestra una ventana con proveedores, seleccione un proveedor y luego un producto y presione el botón Cargar.

3) Presione el botón Guardar.

Modificar renglón
1) Seleccionar un renglón
2) Presionar el botón Modificar Renglón.
3) Se muestra un formulario con la descripción y los adicionales del renglón seleccionado.

4) Modificar los datos y presionar Modificar Renglón.

Eliminar renglón
1) Seleccionar un renglón.
2) Presionar el botón Eliminar Renglón.

Ver Descripciones
1) Seleccionar un renglón.
2) Presionar el botón Ver Descripcines. Esto muestra las descripciones técnicas de los productos del renglón.
3) Si se presiona el boton Guardar
4) Si se presiona el botón Dar OK se verifica que significa que se ha realizado el primer control.
5) Si se presiona Generar documento se genera un documento word para el renglón con las descripciones de los productos.
6) presionar Imprimir para Imprimir el documento generado.

Terminar Licitación
Presionar el botón Terminar
