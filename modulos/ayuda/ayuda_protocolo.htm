Esta página permite cargar los requisitos legales para una licitación.

Datos de la entidad

1) Completar los datos de la entidad ingresar datos para los campos Procedimiento y Lugar. El campo comentario es opcional. Guardar por defecto.
2) Si se selecciona la opción "Guardar por defecto" (click en la casilla de verificación ( ) se guarda el protocolo que se esta cargando por defecto para las próximas licitaciones de la entidad. Requisitos Legales
3) eleccionar los requesitos legales correpondientes para la licitación realizando click en la casilla de verificación ( ) que se visualiza para cada requisito legal, opcionalmente puede ingresar un comentario. Nuevo Requisitos
4) Para ingresar un nuevo requisito legal:
  4.1) Ingrese el nombre del requisito.
  4.2) Seleccione la primera casilla de verificación para que el nuevo requisito quede seleccionado en el protocolo.
  4.3) Puede ingresar opcionalmente un comentario.
  4.4) Seleccione la segunda casilla de verificación para incorporar el nuevo requisito.

Cargar Protocolo

5) Presione el Botón "Cargar Protocolo" para guardar los requisitos chequeados en el protocolo, con lo cuál se genera un archivo que contiene dicho protocolo. Se vuelve a la página que muestra la licitación seleccionada, en la sección de archivos puede seleccionar el archivo que se generó con el protocolo cargado. El nombre del archivo es req_legales_numero.pdf donde "numero" corresponde al número que tiene asignada la licitación seleccionada. Por ejemplo si la licitación tiene como número de licitación 1205 el nombre del archivo será "req_legales_1205.pdf "
