Si la orden esta asociada a una licitación, se muestra una lista con las licitaciones, se elige la opción Presentadas, con esto se obtiene una nueva lista, elegir una de las licitaciones. Se muestra una pantalla donde los datos de la licitación ya fuieron cargados, luego puede ingresar los datos para una nueva orden.

Realizar un orden de compra.
1) Completar la fecha de entrega, seleccionar un proveedor del menú, selecionar un contacto para ese proveedor; también seleccionar el tipo de moneda.
 2) Para completar el cliente presione la palabra Cliente, esto muestra los clientes disponibles. Seleccionar un cliente, esto muestra los datos del cliente seleccionado ( Para buscar un cliente en la lista seleccione la lista de clientes, presione las primeras letras del nombre del cliente, esto posicionará el cursor en el nombre del cliente que coincida con las letras presionadas. ). Si el checkbox con titulo dirección de entrega está tildado se carga la dirección del cliente como dirección de entrega en la orden de compra. En caso de no tildar el checkbox, completar la dirección de entrega. También puede ingresar un nuevo cliente.
3) Para completar la forma de pago puede seleccionar una forma de pago o puede presionar el botón Forma de pago para modificar la forma de pago (cambiar el nombre y marcar Default si se desea que esa forma de pago aparezca en la lista) que se asociará a la orden. Otra opción es ingresar una nueva forma de pago con lo que debera detallar el nombre de esa forma de pago (puede marcar la opción Default con lo cual esa forma de pago se agregará a la lista) que se asociará a la orden.
4) Para añadir productos a la orden de compra presionar el botón Agregar. Esto mostrará una lista con los productos disponibles.
  4.1) Seleccione tipo de producto, luego seleccione un producto y presione Cargar. Al terminar de cargar productos presione el botón Salir.
Nota: Puede cargar todos los productos que se requieren en la orden de compra sin salir de la pantalla que muestra los productos.
5) Si desea eliminar un producto de la orden de compra seleccione el checkbox correspondiente y presione el botón Eliminar.
6) Presione el botón Guardar para guardar la orden de compra. Esta orden se guarda con estado "Pendiente".
