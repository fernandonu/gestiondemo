Esta página permite agregar nuevos productos o editar los datos de los productos existentes.

1) Seleccione el tipo de producto o marque Nuevo tipo de producto con lo cual deberá ingresar un nuevo tipo.
2) Ingrese la Marca del producto.
3) Ingrese el Modelo del producto.
4) Ingrese una Descripción general para el producto.
5) Seleccione un Proveedor para el producto agregado.
7) Seleccione el Precio del producto.
8) Presione el botón Guardar.

Editar datos de un producto
1) Para editar los datos de un producto seleccione tipo de producto.
2) Seleccionar uno de los productos de la lista de Productos cargados.
3) Editar los datos del producto seleccionado.
4) Presionar el botón Guardar.
