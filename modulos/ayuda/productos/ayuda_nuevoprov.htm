Esta página permite cargar nuevos proveedores,editar los datos de los proveedores ya existentes, activar o desactivar proveedores y agregar contactos.

Agregar Proveedor
1) Ingrese Razón Social, cuit y seleccione el tipo de IVA.
2) Seleccione el checkbox con título Activo.
3) Ingrese una observación para el proveedor.
4 ) Presione el botón Agregar/Modificar proveedor.

Modificar proveedor
1) Seleccionar el nombre del Proveedor del combo que despliega los nombres de los proveedores. Esto mostrará los datos del proveedor.
2) Modificar los datos correspondientes.
3) Presionar el botón Agregar/Modificar Proveedor.

Seleccionando uno de los proveedores de la lista puede cambiarse la clasificación del proveedor haciendo click en el botón Clasificar (con esto se muestra la clasificación actual), los cambios realizados se guardan automaticamente.

Requisitos para clasificar un proveedor

* Tiempo de Entrega
* Forma de Pago
* Precio
* Certificado ISO 9001
* Calidad del Producto

Clasificación del proveedor según la cantidad de requisitos que cumplen

A: El proveedor cumple con cinco requisitos
B: El proveedor cumple con cuatro requisitos
C: El proveedor cumple con tres requisitos
D: El proveedor cumple con dos requisitos
E: El proveedor cumple con un requisito

 

Agregar Nuevo contacto
1) Seleccionar el proveedor al cual le agregará un contacto.
2) Presionar el botón Nuevo contacto.
3) Ingresar los datos que se solicitan en la página.
4) Presionar el botón Agregar Contacto.

Modificar/Eliminar contactos para un proveedor
1) Seleccionar el proveedor al cual le agregará un contacto.
2) Presione el botón con el nombre del contacto. Esto mostrará los datos del contacto.
3) Para modificar los datos del contacto edite los datos y presione el botón Cambiar.
4) Para eliminar un contacto presione el botón Eliminar.

Activar Proveedor
1) Seleccionar un proveedor.
2) Chequear el botón Activo.
3) Presionar el botón Agregar/Modificar proveedor.

Desactivar Proveedor
1) Seleccionar un proveedor.
2) Pulsar el botón Activo para que quede sin marcar.
3) Presionar el botón Agregar/Modificar proveedor.
