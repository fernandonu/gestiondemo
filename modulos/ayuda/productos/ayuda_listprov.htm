Este página lista los proveedores con los contactos para dicho proveedor.

Al realizar click en una fila se muestran los datos del conctacto correpondiente se puede agregar, modificar o eliminar contactos del proveedor seleccionado..

Se permite realizar una búsqueda por proveedor. Ingrese el texto a buscar (en el campo buscar) y seleccione en que campo buscar (en el campo en ) y presione el botón Ver Datos.

 

Al realizar un click en la fila que muesta el proveedor; se obtinen los datos del contacto para el proveedor seleccinado y se permite:

Agregar Nuevo contacto
1) Presionar el botón Nuevo contacto.
2) Ingresar los datos correspondientes.
3) Presionar el botón Agregar Contacto.

Ver/modificar/eliminar contactos para un proveedor
1) Presione el botón con el nombre del contacto. Esto mostrará los datos del contacto
2) Para modificar los datos del contacto edite los datos y presione el botón Cambiar.
3) Para eliminar un contacto presione el botón Eliminar.

Clasificar a un proveedor
Esta clasificación corresponde .....
1) Presione el botón Clasificar; se mostrará el detalle del provedor y la clasificación correspondiente.
2) Seleccionar una nueva clasificación (el cambio realizado queda guardado automaticamente).
3) Opcionalmente puede ingresar un comentario; luego presionar Guardar comentario.
