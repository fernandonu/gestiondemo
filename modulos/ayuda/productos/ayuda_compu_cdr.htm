Esta página permite mostrar las órdenes y el detalle de cada una de ellas.

Se permite realizar una búsqueda, ingresando el texto a buscar (en el campo Buscar) y seleccionando en que campo buscar (en el campo En ); presione el botón Buscar. Con el botón Borrar búsqueda se obtiene la lista todas las órdenes.

Haciendo click sobre una de las órdenes se obtiene una descripción de ella, también se obtiene el detalle de las órdenes de produccción , de compras, facturas y remitos asociados a esta orden.
