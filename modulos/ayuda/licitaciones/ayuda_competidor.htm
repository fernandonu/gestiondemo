Esta página permite ingresar un nuevo competidor y sus datos. Se muestra el nombre de los competidores que ya están cargados en la base de datos.
Para cargar un nuevo competidor: si el formulario contiene datos presionar el botón Nuevo para resetear el formulario, ingresar los datos que se requieren en la página y presionar el botón Guardar.

Para eliminar un competidor:
En la tabla que se muestran los competidores seleccionar el competidor a eliminar y presionar el botón Eliminar.

Para modificar un competidor:
Seleccionar el nombre del competidor de la lista de competidores.Modificar los datos y presionar el botón Guardar.
