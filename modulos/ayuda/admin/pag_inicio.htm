Esta página permite completar o cambiar sus datos personales, cambiar la página de inicio del sistema y crear o cambiar accesos directos.

Datos personales:

1) En el formulario con título Datos Personales complete o edite sus datos personales.

2) Presione el botón Guardar.

Los campos de la sección Firma pueden llenarse (opcionalmente) con sus datos personales, estos se mostrarán en el PDF que se crea para las ordenes de compra.

Cambiar página de inicio del sistema

1) En la sección de Página de Inicio: se indica la página que actualmente es la página de inicio del sistema.

2) Presionar el botón Cambiar. Con esto se muestra el mensaje: "seleccione la nueva página de inicio en el menú".

3) Seleccionar del menú la nueva página de inicio. Una vez que seleccionada la nueva página de inicio se muestra la página de Perfil de Usuario.

4) Presionar el botón Guardar.

5) Salir del Sistema e ingresar nuevamente para ver el cambio realizado.

Agregar/ Cambiar accesos directos

1) Presionar el botón Agregar cuando no esta disponible el acceso directo; sino presionar el botón Cambiar.

Con esto se muestra el mensaje: "seleccione la nueva página de inicio en el menú".

2) Seleccionar del menú el nuevo acceso directo. Una vez seleccionada el nuevo acceso directo, se muestra la página de Perfil de Usuario.

3) Presionar el botón Guardar.

4) Salir del Sistema e ingresar nuevamente para ver el cambio realizado.
